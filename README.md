📊 Análise de dados: Reduzindo o Cancelamento de Clientes
Este projeto foi desenvolvido durante a Jornada Python da Hashtag Treinamentos. O foco principal é realizar uma análise exploratória de dados para identificar os motivos pelos quais os clientes de um serviço de assinatura estão cancelando e propor soluções estratégicas para aumentar a retenção.

🎯 Objetivo do Projeto
Identificar os principais pontos de atrito na jornada do cliente e entender o perfil dos usuários que cancelam o serviço, visando reduzir a taxa de cancelamento por meio de decisões baseadas em dados.

🛠️ Tecnologias e Bibliotecas Utilizadas
Python: Linguagem principal.

Pandas: Para manipulação, limpeza e tratamento da base de dados.

Plotly: Para criação de gráficos interativos que facilitam a visualização de padrões.

Openpyxl / Nbformat: Bibliotecas auxiliares para leitura de arquivos Excel e suporte ao ambiente Jupyter.

📈 Etapas do Processo
Importação de Dados: Leitura da base de dados de clientes.

Tratamento de Dados: Limpeza de valores vazios e remoção de colunas inúteis para a análise.

Análise Inicial: Cálculo da taxa de cancelamento atual.

Análise Profunda: Criação de dashboards e gráficos comparativos para identificar a causa raiz dos cancelamentos (ex: duração do contrato, ligação callcenter, dias de atraso).

💡 Aprendizados e Insights
Uso do Plotly Express para criar visualizações rápidas e intuitivas.

Importância do pré-processamento de dados antes de iniciar qualquer análise estatística.

Identificação de que planos mensais e problemas com o suporte técnico costumam ser os maiores vilões do cancelamento.
